# Roadmap

## Milestones

### M1: Baseline Platform
- ARC deployed
- Runner Scale Set created
- Smoke workflow running successfully

### M2: Custom Runner Image
- Custom image built and versioned
- Image scanning and release process defined
- Workflows updated to use the image

### M3: Observability and Guardrails
- Dashboards and alerts defined
- Runbook and playbook validated

### M4: Governance
- Define which repos/workflows can use self-hosted runners
- Document standards for labels, permissions, and secrets

## Definition of Done
- Runners are stable under load.
- Incident response procedures exist and are tested.
- Upgrade/rollback procedures exist.
